Есть три базовых канала (Channel1, Channel2, Channel3), на них подписаны все пользователи автоматически. Чтобы создать новый канал, нужно на сервере написать: 
create "Your Channel"
Каждый пользователь может подписаться на новый канал, для этого он должен знать имя этого канал, после подписки, у него появится этот канал в подписках.
Чтобы покинуть комнату следует написать
/quit
Имена пользователей и названия комнат уникальны для всей системы.
